This bot is designed to easily track daily in-game activity. It automatically provides up-to-date information so players don't miss important events and can plan their gameplay more effectively.
The bot can be added to a chat or used in private messages.
===
Этот бот предназначен для удобного отслеживания ежедневной активности в игре. Бот автоматически предоставляет актуальную информацию, чтобы игроки не пропускали важные события и могли эффективнее планировать игровой процесс.
Бота можно добавить в беседу или пользоваться в ЛС.
